
# EcoPay Backend

A backend build in python django rest framework



## Creating virtual env

```bash
python -m venv venv
```


## Downloading Dependencies

```bash
pip install -r requirements.txt
```
## Running on Localhost

from the root of the folder


```bash
python manage.py runserver
```
